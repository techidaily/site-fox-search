## Custom Properties

This view allows you to set IIS properties, such as attributes and collection items, that you can normally see and modify using IIS Manager's Configuration Editor.

![Important](https://cdn.advancedinstaller.com/svg/common/IconMessageInfo.svg)Supported only for IIS 7 or newer. If the setup package is installed on a machine with an older IIS version these settings will be ignored.

![Custom properties](https://cdn.advancedinstaller.com/img/ui/custom-properties.png "Custom properties")
